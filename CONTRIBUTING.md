# Contributing to EquipCompare (El Séquito Edition)

Este es un fork especializado en la comparativa de equipo para Turtle WoW.

## Reglas Técnicas
1. **Tooltip Hooking**: Cualquier modificación en la lógica de enganche de tooltips debe realizarse en `EquipCompare.lua`.
2. **Lua 5.0**: Evitar operadores modernos.

## Desarrollo
Forkea, mejora y abre un PR en GitHub.
