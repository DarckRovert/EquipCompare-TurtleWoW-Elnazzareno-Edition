# Contributor Covenant Code of Conduct

Como miembros de **El Séquito del Terror**, nos comprometemos a mantener una conducta profesional en la comparación y distribución de loot.

## Compromiso
- Objetividad al comparar equipo.
- Respeto a las reglas de Loot del clan.
- Colaboración técnica para mejorar los algoritmos de comparación.
