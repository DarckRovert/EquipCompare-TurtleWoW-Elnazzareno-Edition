# ❓ Wiki: FAQ [El Séquito del Terror] — EquipCompare

Preguntas frecuentes y resolución de problemas técnicos para la utilidad de equipo v9.4+.

## 🛠️ Error: No veo la comparativa de mi abalorio/anillo.
- **Causa**: Al tener dos abalorios o anillos, EquipCompare podría comparar solo contra el primer slot por defecto.
- **Solución**: Mantén pulsada la tecla `Alt` para alternar la comparativa entre el primer y segundo slot equipado del mismo tipo.

## ⚙️ ¿Cómo activo o desactivo el sistema de comparativa rápidamente?
- **Control**: EquipCompare v9.4.0 puede configurarse para mostrarse siempre o solo al pulsar una tecla. Escribe `/ec` para abrir las opciones y activa el modo **Override Shift** para que solo aparezcan las barras Diamond Tier al pulsar `Shift`.

## ⚡ ¿Por qué los tooltips secundarios no tienen bordes pfUI?
**Estado**: ✅ Corrección en v9.4.0.
- Hemos rediseñado el frame `EquipCompareTooltip` para que herede directamente las texturas de borde de pfUI, manteniendo la coherencia estética del clan Séquito del Terror.
